## 長い説明

```
Climb Log は、ボルダリングの登攀をかんたんに記録できるアプリです。
通うジムごとに課題を記録し、完登率や苦手な壁を見える化して上達につなげましょう。

【主な機能】
■ 登攀記録
・ジム / グレード / 壁の種類 / トライ数 / 完登 を記録
・課題の写真とベータ（攻略メモ）を保存
・日付ごとのリスト表示とカレンダー表示

■ 統計（プレミアム）
・月別の完登数の推移
・グレード別の完登率
・壁の種類別の完登率（苦手な順）

■ その他
・完全オフラインで動作。データは端末内に保存
・記録のCSVエクスポート（プレミアム）

【プレミアムについて】
・統計ダッシュボード、写真の無制限保存、CSVエクスポートが解放されます。
・サブスクリプション / 買い切りでご利用いただけます。

ジム通いのモチベーション管理に、ぜひお使いください。
```

## プロモーションテキスト（App Store, 170字以内）

```
通うジムごとに課題を記録して、完登率や苦手な壁を分析。写真とベータも残せて、ジム通いの上達がはっきり見えるボルダリング記録アプリです。
```
