# プライバシーポリシー（Climb Log）

最終更新日: 2026-06-20

本プライバシーポリシーは、Climb Log（以下「本アプリ」）における利用者情報の取り扱いについて定めるものです。

> 注: ストア申請には「公開URL」が必要です。本ファイルを GitHub Pages 等で公開し、そのURLを App Store Connect / Google Play Console に登録してください。事業者名・連絡先は実際の情報に置き換えてください。

## 1. 収集する情報と利用目的

本アプリが扱う情報は次のとおりです。

- **登攀記録・写真・メモ**: 利用者が入力したジム、課題、グレード、写真等のデータ。これらは**利用者の端末内にのみ保存**され、開発者が収集・送信することはありません。
- **カメラ / 写真ライブラリ**: 登攀記録に写真を添付するためにのみ使用します。取得した写真は端末内に保存されます。
- **購入情報**: プレミアム機能の購入・復元のため、課金基盤（RevenueCat および Apple / Google の課金システム）を通じて購入状態を管理します。これらの事業者により、購入識別子等が処理されます。

本アプリは、氏名・メールアドレス・位置情報などの個人を特定する情報を独自に収集しません。

## 2. 第三者サービス

本アプリは以下の第三者サービスを利用します。各サービスの取り扱いは各社のポリシーに従います。

- RevenueCat（課金管理） — https://www.revenuecat.com/privacy/
- Apple App Store / Google Play（課金処理）

## 3. データの共有・第三者提供

開発者は、利用者の登攀記録や写真を第三者に提供・販売しません。上記の課金処理に必要な範囲を除き、外部へ送信しません。

## 4. データの保存と削除

登攀データは端末内に保存されます。アプリをアンインストールすると、端末内のデータは削除されます。

## 5. 子どものプライバシー

本アプリは特定の年齢層を対象とした情報の収集を行いません。

## 6. 本ポリシーの変更

本ポリシーは必要に応じて改定されることがあります。重要な変更がある場合はアプリ内またはストア上で通知します。
